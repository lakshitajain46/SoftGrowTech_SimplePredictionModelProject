## Simple Prediction Model using Machine Learning

### Project Overview
This project demonstrates a simple Machine Learning prediction model using Linear Regression. The model predicts house prices based on house area.

## Tools & Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Machine Learning Algorithm
Linear Regression

## Features of Project
- Dataset creation
- Data visualization
- Model training
- Price prediction
- Error calculation

## Learning Outcomes
- Understanding Machine Learning basics
- Training prediction models
- Working with Linear Regression
- Making predictions using Scikit-learn
